# 加强削弱专项做题方法学习笔记

## 核心目标

完整还原作者解加强削弱（含前提型）题的思维流程，涵盖“审题→梳理论证结构→分析选项→力度比较→验证答案”全流程，明确每一步的操作技巧、判断依据及易错点，确保可独立复现方法并迁移到同类题型。

## 第一章 考勤与题型定位

### 1.1 题量分布特征

拿到国考行测卷，首先明确加强削弱题型的题量差异：

- **地市级/执法卷**：含“一拖五”题型，加强削弱（包含前提型）共5题；

- **副省级卷**：无“一拖五”题型，加强削弱题量多于地市级/执法卷。

作者强调：无论哪类试卷，加强削弱都是“严谨模块”，方法固定，掌握后可稳定拿分，无需过度预测考勤，重点在方法本身。

### 1.2 前提型题型核心定位

前提型是国考“送分题”，核心特征如下：

- **必考规律**：除去年地市级卷外，近数年所有试卷均考1道；

- **考查重点**：99%概率考“搭桥”，几乎不考其他前提类型（如“镁离子”类题目国考不涉及）；

- **解题优先级**：拿到前提型题，直接优先找“搭桥项”，无需考虑其他选项，节省时间。

## 第二章 前提型解题方法：聚焦“搭桥”与反向验证

前提型题的核心是“补全题干论证的逻辑断层”，作者将解题方法分为两类，需根据题干是否存在“概念跳跃”选择对应方法。

### 2.1 搭桥法：适用于“概念跳跃”类题干

#### 2.1.1 搭桥法操作步骤（作者核心流程）

1. **第一步：定提问方式**：确认题干问法为“前提”“假设”“必要条件”，明确为前提型题；

2. **第二步：找论点、论据**：通过“因此”“所以”“有人认为”等标志词拆分题干，定位论据（支持结论的依据）和论点（最终观点）；

3. **第三步：找概念跳跃**：对比论据和论点中的核心概念，找出未直接关联的“断层概念”（记为A→B的论据、A→C的论点，断层为B与C）；

4. **第四步：选搭桥项**：寻找直接连接“断层概念”的选项（即B→C），此类选项即为前提。

### 2.2 反向验证法：适用于“无概念跳跃”类题干

#### 2.2.1 反向验证法操作步骤（作者核心流程）

1. **判断适用场景**：若题干论点与论据无明显概念跳跃（如论据和论点均围绕同一核心话题），则用反向验证；

2. **核心逻辑**：前提是“必要条件”（没它不行），因此否定选项后，若题干结论不成立，则该选项为前提；若否定后结论仍可成立，则排除；

3. **操作细节**：逐一否定选项，仅需验证“否定后结论是否必然失效”，无需分析其他，快速排除干扰项。

3. **关键提醒**：反向验证时，若选项包含“且”关系（如“需A且B”），否定后为“非A或非B”，只要有一个条件满足结论仍成立，该选项就不是必要前提——这是常见易错点，需重点关注。

## 第三章 加强削弱型：三类核心论证结构与解题逻辑

加强削弱型题的核心是“识别论证结构”，作者将国考高频结构分为三类：因果共存型、知果求因型、知因求果型，每类结构的削弱/加强方式固定，需精准匹配。

### 3.1 因果共存型：“A与B同时存在，推测A→B”

#### 3.1.1 结构特征（作者判断依据）

- 题干先描述“两个现象同时发生”（如“工具使用增多”与“人类脸型变化”同时出现）；

- 结论直接推测“其中一个现象是另一个的原因”（如“工具使用导致脸型变化”）；

- 核心漏洞：仅通过“共存”就定因果，未排除“B→A”“C→A和B”等可能性。

#### 3.1.2 削弱/加强方式（作者核心总结）

|类型|操作逻辑|关键要求（作者强调）|
|---|---|---|
|削弱|优先“他因削弱”：找“与A共存的C，且C→B”|他因需与A“同时存在”（如时间、主体一致）|
|加强|优先“解释因果”：说明A如何导致B|需补全A到B的逻辑链条，而非仅重复A或B|
#### 3.1.3 例题1：工具使用与人类脸型变化

```Markdown

```

作者分析过程：

1. **定结构**：先找共存现象（A与B同时出现），再看结论为“A→B”，确定为因果共存型；

2. **找他因削弱**：需找“与A同时存在的C，且C→B”；

3. **选项逐一拆解**：

    - 选项B：“两百万年前（与A的时间一致），人类食物类型发生变化（C），且食物变化会影响脸型（C→B）”——满足“他因与A共存”，此时无法确定是A还是C导致B，削弱结论；

    - 选项A：“与人类接近的灵长类动物未使用工具，脸型也变化”——主体不一致（灵长类≠人类），不当类比，排除；

    - 选项C：“石器工具使食物更易咀嚼”——仅提A的作用，未提B（脸型），无关，排除；

    - 选项D：“早期人类大脑体积增大”——与A（工具）、B（脸型）均无关，排除；

4. **答案逻辑**：只有选项B满足“他因与A共存+他因→B”，是唯一有效削弱项。

作者分析过程：

1. **定结构**：共存现象（A与B）+结论“A→B”，因果共存型；

4. **易错点提醒**：若选项仅提“C→B”但未说明“C与A共存”（如“某时期大型食肉动物导致反刍动物减少”，未提与人类活动同期），则不是他因削弱，而是无关项——**时间/主体一致是他因削弱的核心前提**。

### 3.2 知果求因型：“已知B，推测原因是A”

#### 3.2.1 结构特征（作者判断依据）

- 题干先给出“一个结果/现象B”（如“小果占比过大”“贝类分泌红霉素”）；

- 结论推测“导致B的原因是A”（如“小果占比大是因为套带藻”“红霉素来自体内微生物”）；

- 核心漏洞：未排除“其他原因导致B”，削弱优先找“他因”，加强优先“排除他因”。

#### 3.2.2 削弱/加强方式（作者核心总结）

|类型|操作逻辑|关键要求（作者强调）|
|---|---|---|
|削弱|他因削弱：找“除A外，能导致B的其他原因C”|C需与A“共存”（即A存在时C也存在）|
|加强|1. 排除他因：说明无其他原因导致B；2. 确认A→B：解释A如何导致B|避免“不明确选项”（如“可能”“不一定”）|

作者分析过程：

1. **定结构**：已知B（小果占比大），推测因A（套带藻），知果求因型；

2. **找他因**：需找“与A共存的C，且C→B”；

    - 选项D：“今年未追加施肥（C），导致果树营养不足（C→B：营养不足使果实变小）”——C（未施肥）与A（套带藻）同期存在，且C能直接导致B，是有效他因削弱；

    - 选项A、B：“小果甜度与大果一致”“套带藻仅附着在枝叶上”——均与“小果占比大（B）”无关，排除；

4. **关键提醒**：“不明确选项”（如含“可能”“不一定”“尚未可知”）在知果求因型中一律排除，因为其无法确定是否影响B，力度远弱于明确的他因。

    - 选项D：“其他贝类也能分泌红霉素”——主体不一致（题干是“某贝类”，不是“其他贝类”），排除。

### 3.3 知因求果型：“已知A，推测结果B”

#### 3.3.1 结构特征（作者判断依据）

- 题干先给出“一个原因/条件A”（如“喝牛奶”“投电梯广告”）；

- 结论推测“A会导致结果B”（如“喝牛奶助眠”“电梯广告提高利润”）；

- 核心漏洞：未考虑“A存在时，B是否必然发生”，削弱优先“有因无果”，加强优先“解释A→B”。

#### 3.3.2 削弱/加强方式（作者核心总结）

|类型|操作逻辑|关键要求（作者强调）|
|---|---|---|
|削弱|有因无果：找“A存在，但B不发生”的情况|需保证“A的条件一致”（如同一主体、同一时间）|
|加强|解释因果：说明A如何导致B（补全逻辑链条）|需明确A与B的直接关联，避免间接推导|
#### 3.3.3 例题1：电梯广告与利润


作者分析过程：

1. **定结构**：已知A（投电梯广告），推测B（利润涨），且题干提“A后B发生”，需注意“相关性≠因果性”，知因求果型；

2. **找有因无果**：需找“投电梯广告（A），但利润未涨（非B）”的情况，且需保证“与题干门店条件一致”；

3. **选项逐一拆解**：

    - 选项B：“与该门店规模一致的S店（条件一致），投电梯广告后（A），利润反而下降（非B）”——完美符合“有因无果”，直接削弱“ A→B”的因果关系；

    - 选项A：“该门店利润比同期未投广告的H店高”——错误在于“与他人比”：H店利润低可能有其他原因（如位置差），不能说明A导致B；正确对比应是“该门店投广告前后的利润对比”，排除；

    - 选项C：“该门店同时投了报纸广告”——仅提“其他广告”，未提“利润是否由电梯广告导致”，不明确，排除；

4. **对比实验原则**：知因求果型的削弱/加强中，“同一主体前后对比”或“条件一致的主体对比”才有效，“不同条件主体对比”（如A选项的H店）是常见陷阱，需警惕。

## 第四章 国考高频模型题：固定套路与秒杀技巧

除三类核心结构外，作者还总结了国考常考的“模型题”，这类题有固定解题套路，可快速秒杀。

### 4.1 根本原因型：“原观点A→B，反对者C→B”

#### 4.1.1 模型特征（作者判断依据）

- 题干存在两个观点：

    - 原观点：A是导致B的原因（如“情绪波动导致哮喘”）；

    - 反对者：C是导致B的原因（如“气道炎症导致哮喘”）；

- 提问方式：“最能削弱反对者观点”或“最能支持原观点”。

#### 4.1.2 解题套路（作者核心秒杀技巧）

⚠️ 重要提醒：削弱反对者=支持原观点，核心是“找A→C”的选项（即A是C的原因，A是根本原因）；支持反对者=削弱原观点，核心是“找C→A”的选项（即C是A的原因，C是根本原因）。

3. **选项锁定**：选项D：“消极情绪（A）会提高气道炎症（C）的发生率”——直接说明A是C的原因，因此B的根本原因是A，削弱反对者；

4. **秒杀逻辑**：无需分析其他选项，只要找到“ A→C”的选项，直接当选，这是该模型的固定套路。

### 4.2 统计样本型：“通过样本数据推整体结论”

#### 4.2.1 模型特征（作者判断依据）

- 题干通过“某类样本的调查数据”（如“参与冰雪旅游调查的人”）推导“整体情况”（如“冰雪旅游需求高”）；

- 核心漏洞：样本是否具有代表性（是否存在“幸存者偏差”）。

#### 4.2.2 解题套路（作者核心秒杀技巧）

- 削弱：指出“样本不具代表性”（如“样本仅包含某类人群，无法代表整体”）；

- 加强：指出“样本具有代表性”（如“样本随机选取，覆盖各类人群”）。

### 5.2 易错点总结（作者反复强调）

1. **混淆“不明确选项”与“有效选项”**：

2. **对比实验的“条件一致性”**：

3. **“不能支持”题的答案倾向**：
